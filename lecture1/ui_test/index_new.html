📐 Flexbox 핵심 속성

• display: flex→ 컨테이너를 flexbox로 설정하여 내부 요소들을 유연하게 배치
• justify-content: space-between→ 로고와 메뉴를 양 끝으로 정렬
• align-items: center→ 세로축 중앙 정렬로 모든 요소를 수직 가운데 배치
• gap: 15px→ 메뉴 항목들 사이에 15px 간격을 자동으로 추가
💡 TIP: Flexbox는 반응형 레이아웃을 만들 때 가장 많이 사용되는 CSS 레이아웃 방식입니다!
